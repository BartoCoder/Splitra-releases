# Splitra — Releases

This repository hosts official release binaries for **Splitra**, a file
splitter and merger application developed by **Core Layer**.

## Contents

- Windows installers (`.exe`)
- Linux installer archives (`.tar.gz`)
- Android packages (`.apk`)

## Download

Official downloads and update information are available through the
Splitra application itself, or via the project's update manifest.
